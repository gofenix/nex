# Guestbook - Nex Framework Demo

A simple guestbook application built with Nex framework and HTMX.

## Features

- Sign guestbook with name and message
- View all messages
- Delete messages
- Real-time updates with HTMX

## Run

```bash
cd examples/guestbook
mix deps.get
mix nex.dev
```

Visit http://127.0.0.1:4001
