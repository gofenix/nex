# Chatbot - Nex Framework Demo

A simple AI chatbot built with Nex framework and HTMX.

## Features

- Real-time chat interface
- Streaming AI responses
- Chat history persistence

## Setup

```bash
cd examples/chatbot
cp .env .env.example  # Edit with your API key
mix deps.get
mix nex.dev
```

Visit http://127.0.0.1:4002
