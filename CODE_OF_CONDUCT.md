# Code of Conduct

## Our Pledge

We want the Nex community to be open, respectful, practical, and welcoming.

Contributors, maintainers, and participants are expected to make this project a harassment-free experience for everyone, regardless of background, identity, or experience level.

## Expected Behavior

Examples of positive behavior include:

- being respectful and constructive
- assuming good intent while giving clear feedback
- focusing on ideas, code, and outcomes instead of personal attacks
- welcoming newcomers and helping them navigate the project
- accepting that reasonable people may disagree on trade-offs

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, abuse, or discriminatory language
- trolling, intimidation, or personal attacks
- publishing private information without permission
- repeated bad-faith disruption of discussions or reviews
- hostile or demeaning conduct in issues, pull requests, or community spaces

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues
- pull requests
- discussions
- social posts representing the project
- direct interactions connected to project collaboration

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer.

Please include context, links, screenshots, and any other information that helps evaluate the report.
